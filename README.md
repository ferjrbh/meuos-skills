# meuos-skills — Loja de Skills do MeuOS (privada)

Catalogo de skills do MeuOS para alunos, distribuido como **plugin do Claude Code** via marketplace.

## Instalar (no Claude Code do aluno)

    /plugin marketplace add ferjrbh/meuos-skills
    /plugin install meuos-skills@meuos

Depois rode `/reload-plugins`. As skills ficam como `/meuos-skills:<nome>` e o Claude as usa sozinho quando o pedido casa.
